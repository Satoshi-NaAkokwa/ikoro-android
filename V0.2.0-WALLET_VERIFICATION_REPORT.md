# Ikoro Android v0.2.0-wallet — Wallet-Only Rebuild

**Project:** Ikoro Android  
**Version:** 0.2.0-wallet  
**Date:** 2026-06-18  
**Scope:** Discarded full-app rebuild; wallet-only release.

---

## What was removed

- **Chat** (Simplex/Nostr DM layer, contact DB, chat UI)
- **Calls** (LiveKit WebRTC, camera/audio permissions, `RoomActivity`)
- **Market** (thirdweb integration, P2P/market UI, market contracts UI)
- **Identity verticals** (credentials, token issuance, detailed identity screen)
- **Notifications** foreground service, boot receiver, ntfy service
- **Unused dependencies:** LiveKit, nostr-sdk, thirdweb Connect, zxing QR, Room DB, Lottie, DataStore, biometric, SSE OkHttp

## What remains

- Self-custodial 24-word BIP-39 identity creation and restore
- TrustWalletCore native library for address derivation (Bitcoin + EVM chains)
- Wallet tab with send/receive for Rootstock, Ethereum, Base, Polygon, Arbitrum, Optimism, BSC, Bitcoin
- `IdentityManager` injected into `WalletManager` to avoid main-thread KeyStore work
- Crash guard in `WalletScreen`
