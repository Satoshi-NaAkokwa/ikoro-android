## iOS Emulator Feasibility

- **iOS Simulator:** ❌ Not possible on Linux. Requires macOS + Xcode.
- **Android Emulator:** VPS lacks nested virtualization (`/dev/kvm` missing, no vmx/svm). Android emulator can be installed but will run via slow software emulation. Not practical for E2E testing.

## Blockers for full certification

1. Real Android device or KVM-capable machine for interactive testing.
2. Deployed Rootstock contracts and addresses inserted into `BuildConfig`.
3. Real wallet with testnet/mainnet balance to exercise send/receive.
4. Running Agbara bot service for DM delivery verification.
5. Two devices/accounts to verify LiveKit calls.

## Honest certification

This build is **code-complete and build-verified** for Phase 1 + stub completion. It is **not yet interactively verified** on a real device. I will not certify any end-to-end flow that has not been exercised.
